# FileIyagi (파일이야기) v1.3.0

Windows와 Linux에서 사용할 수 있는 **빠르고 가벼운 파일 관리자**입니다.

빠르고 반응성이 뛰어난 파일 관리자로, **한글 파일이름 변경과 검색 시 텍스트 입력이 안정적**으로 작동하도록 설계되었습니다.

고속 탐색, 키보드 중심 조작, 부드러운 썸네일 보기를 제공합니다.
인터페이스는 시스템 언어에 따라 자동으로 전환되어 자연스러운 사용 경험을 제공합니다.

## ✨ 주요 기능

### 탐색
* **브레드크럼 경로 표시줄** — 경로를 블록으로 표시, 클릭으로 상위 폴더 이동
* **경로 직접 입력** — `Ctrl+L`로 편집 모드 전환, `~` 홈 디렉토리 지원
* **경로 복사** — 현재 폴더 블록 클릭 시 전체 경로 클립보드 복사
* **앞으로/뒤로** — 마우스 4·5번 버튼, 툴바 ←→ 버튼으로 히스토리 탐색
* **즐겨찾기 사이드바** — 홈·바탕화면·문서 등 기본 폴더 + 사용자 북마크
* **장치 섹션** — 드라이브·USB 마운트 자동 표시 (볼륨 라벨 우선, Windows·Linux 공통)
* **마운트 자동 감지** (Linux) — USB 연결·해제 시 사이드바 자동 갱신
* **내 장치** — 사이드바에서 "내 장치"를 클릭하면 마운트되지 않은 드라이브까지 전체 목록 표시, 클릭으로 즉시 마운트
* **최근 파일 (RECENT)** — 폴더별 최근 열람 파일 목록, 탐색 시 자동 갱신

### 보기
* **자세히 보기** — 이름·크기·확장자·수정일 컬럼, 폴더 우선 정렬, 컬럼 클릭 정렬
* **아이콘 보기** — 소(96px)·중(150px)·대(224px) 세 단계, 썸네일 자동 표시
* **썸네일** — 이미지(JPEG·PNG·GIF 등) 및 동영상(`ffmpegthumbnailer`) 자동 생성
* **폴더별 보기 모드 저장** — 폴더마다 마지막 보기 모드 기억
* **숨김 파일 토글** — `Ctrl+H` 또는 "." 버튼
* **폰트 크기 조절** — `Alt+휠`로 실시간 조절 (7~24pt)
* **보기 모드 전환** — `Ctrl+휠` 또는 `Ctrl+1~4`
* **정렬 버튼** — 상태표시줄의 이름·크기·확장자·날짜 버튼으로 빠른 정렬 전환

### 검색
* **항상 표시 검색창** — 툴바 오른쪽에 항상 활성화
* **하위 폴더 재귀 검색** — 현재 폴더 기준 전체 하위 탐색
* **300ms 디바운스** — 입력 즉시 실행하지 않고 자연스럽게 시작
* **검색 중 파일 열기** — 검색 결과에서 파일을 열어도 검색 유지
* **검색 결과 드래그** — 검색 결과 파일을 다른 앱으로 드래그 가능

### 파일 작업
* **복사 (F5)** — 즐겨찾기·장치 사이드바가 포함된 대상 폴더 선택 다이얼로그
* **이동 (F6)** — 즐겨찾기·장치 사이드바가 포함된 대상 폴더 선택 다이얼로그
* **삭제 (Del, F8)** — 확인 대화상자 후 삭제; `Shift+Del`은 확인 없이 즉시 삭제
* **이름 변경 (F2)** — 인라인 편집, 확장자 제외 이름만 자동 선택
* **새 폴더 (F7)** — 어디서나 (사이드바 포커스 포함)
* **새 파일** — 현재 폴더에 빈 파일 생성 (툴바 버튼 또는 우클릭 메뉴)
* **잘라내기 (Ctrl+X)** — 파일을 반투명으로 표시; 붙여넣기 시 이동
* **복사 (Ctrl+C)** — 파일을 클립보드에 복사
* **붙여넣기 (Ctrl+V)** — 잘라내기한 파일은 이동, 복사한 파일은 복제
* **다른 앱으로 열기** — MIME 기반 앱 목록 + 추천 앱; 마지막 사용 앱 기억
* **파일 뷰어 (F3)** — 텍스트·이미지 전용 읽기 전용 내장 뷰어
* **파일 편집기 (F4)** — 내장 텍스트 편집기; 선택 없이 F4 누르면 새 파일 생성
* **파일 비교 (Ctrl+D)** — 두 번째 파일 선택 후 File A / File B / Diff 3탭 비교 뷰
* **빠른 미리보기 (F11)** — 선택 파일의 이미지·텍스트를 슬라이드인 패널로 표시
* **터미널 열기 (F9)** — 현재 폴더에서 터미널 실행
* **드래그 앤 드롭** — 파일을 다른 앱(이미지 뷰어, GIMP 등)으로 드래그; 드래그 고스트가 실제 컬럼 너비에 맞춤
* **속성** — 파일 이름·경로·크기·수정일; 폴더는 직접 파일 수 + 전체 재귀 파일 수 표시; 권한(chmod) 변경 가능
* **압축 (Ctrl+우클릭 → 압축...)** — zip·tar.gz·tar.bz2·tar.xz 형식 지원
* **압축 해제** — 아카이브 더블클릭 시 이름 폴더 생성 후 자동 해제·이동

### 시스템
* **창 크기·위치 저장** — 닫으면 저장, 다음 실행 시 복원
* **즐겨찾기 저장** — OS 표준 설정 경로에 저장 (Windows: %APPDATA%, Linux: ~/.config)
* **실시간 파일 감지** (Linux) — inotify 기반, 새 파일 즉시 목록 반영
* **실행 파일 감지** — ELF 매직 바이트 + 쉬뱅(#!) 검사 (NTFS 드라이브에서도 정확히 동작)
* **44개 언어 UI** — 시스템 로케일에 맞는 자동 번역 인터페이스

## 🖧 썸네일 설정

동영상 썸네일을 사용하려면 `ffmpegthumbnailer`를 설치하세요.

```bash
# Linux
sudo apt install ffmpegthumbnailer
```
