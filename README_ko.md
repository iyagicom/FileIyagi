# FileIyagi (파일이야기) v1.52.0

Windows와 Linux에서 사용할 수 있는 **빠르고 가벼운 파일 관리자**입니다.

빠르고 반응성이 뛰어난 파일 관리자로, **한글 파일이름 변경과 검색 시 텍스트 입력이 안정적**으로 작동하도록 설계되었습니다.

고속 탐색, 키보드 중심 조작, 부드러운 썸네일 보기를 제공합니다.
인터페이스는 시스템 언어에 따라 자동으로 전환되어 자연스러운 사용 경험을 제공합니다.

---

## ✨ 주요 기능

### 탐색
* **브레드크럼 경로 표시줄** — 경로를 블록으로 표시, 클릭으로 상위 폴더 이동; 빈 공간 클릭 시 경로 직접 입력 모드 전환; 창이 좁아지면 왼쪽(루트)부터 잘리고 현재 폴더가 항상 보임
* **단일클릭 폴더 열기** — 툴바 ① 토글 버튼; 활성화 시 폴더를 한 번 클릭만 해도 이동
* **경로 직접 입력** — `F12`로 편집 모드 전환, `~` 홈 디렉토리 지원
* **경로 복사** — 현재 폴더 블록 클릭 시 전체 경로 클립보드 복사
* **파일 경로 크럼** — 파일을 단일 선택하면 경로 표시줄 끝에 청록색 파일명 블록이 추가됨 (최대 8글자 표시, 툴팁에 전체 경로); 클릭 시 전체 경로+파일명 클립보드 복사; 폴더 이동 또는 선택 해제 시 자동 사라짐
* **앞으로/뒤로/상위** — 마우스 4·5번 버튼, 툴바 ⇦ ⇧ ⇨ 버튼으로 히스토리 탐색
* **즐겨찾기 사이드바** — 홈·바탕화면·문서 등 기본 폴더 + 사용자 북마크
* **장치 섹션** — 드라이브·USB 마운트 자동 표시 (볼륨 라벨 우선, Windows·Linux 공통)
* **마운트 자동 감지** (Linux) — USB 연결·해제 시 사이드바 자동 갱신
* **내 장치** — 사이드바에서 "내 장치"를 클릭하면 마운트되지 않은 드라이브까지 전체 목록 표시, 클릭으로 즉시 마운트
* **EXT4 파티션 탐색** (Windows 전용) — Windows에 연결된 Linux 디스크의 EXT4 파티션을 관리자 권한으로 직접 탐색 (읽기 전용)
  * `Ctrl+1~5` 보기 모드 완전 지원 (자세히·목록·소/중/대 아이콘)
  * `"."` 버튼으로 dot 파일 숨김/표시 전환
  * 우클릭 **Copy to...** 또는 `Ctrl+C`로 파일을 Windows 폴더로 추출·복사
  * 파일을 드래그해 다른 앱으로 직접 전송 (자동 임시 추출)
  * 경로창에 현재 Linux 경로 표시, 세그먼트 클릭으로 상위 이동
* **최근 파일 (RECENT)** — 폴더별 최근 열람 파일 목록, 탐색 시 자동 갱신
* **최근 경로 히스토리** — 경로표시줄 드롭다운에 최근 방문한 경로 저장; 앱 재시작 후에도 유지

### 다중 패널 (2·3·4창 모드)
* **듀얼 패인 토글** — 상태바 맨 오른쪽 ◫ 버튼(`F1`); 왼쪽=현재 패널, 오른쪽=홈(~/) 시작
* **3·4창 체인 분할** — 2번째 패널의 ◫ 버튼으로 3창, 3번째 패널의 ◫ 버튼으로 4창까지 확장; 버튼을 끄면 그 오른쪽 패널들이 정리되고, `F1`로 2창을 끄면 전부 정리
* **패널 수별 비율·경로 저장** — 2·3·4창 각각의 분할 비율과 각 패널 경로를 따로 저장·복원
* **활성 패널 확대 (옵션)** — 설정 → "다중 패널"에서 켜면 3·4창에서 포커스한 패널이 나머지의 1.5배 폭으로 자동 확대 (기본 꺼짐)
* **Ctrl+S 창 크기 연동** — 창 크기 3단계 순환이 3·4창에서는 패널 수에 비례해 더 넓게 (화면 크기 이내)
* **마지막 경로 저장·복원** — 각 패인의 마지막 방문 경로를 세션 간에 유지
* **2창 비율 저장·복원** — 스플리터 비율을 독립적으로 저장; 1창으로 종료 후 다시 2창으로 전환해도 마지막 2창 비율로 복원
* **세션 간 상태 유지** — 2창 ON/OFF 상태가 앱 재시작 후에도 그대로 복원

### 보기
* **자세히 보기 (Ctrl+1)** — 이름·크기·확장자·수정일 컬럼, 폴더 우선 정렬, 컬럼 클릭 정렬
* **목록 보기 (Ctrl+2)** — 24px 아이콘 + 파일명 단일 열 세로 목록
* **아이콘 보기** — 소(Ctrl+3, 96px)·중(Ctrl+4, 150px)·대(Ctrl+5, 224px) 세 단계, 썸네일 자동 표시
* **썸네일** — 이미지(JPEG·PNG·GIF·WebP 등), 동영상(`ffmpegthumbnailer`), **PDF(첫 페이지 자동 렌더링)** 자동 생성; 디스크 캐시된 썸네일은 스크롤·탐색 중에도 즉시 표시
  * Linux: poppler-qt6 사용
  * Windows: Qt6::Pdf(QPdfDocument) 사용 — ffmpegthumbnailer 없이도 PDF 썸네일 표시
* **폴더별 보기 모드 저장** — 폴더마다 마지막 보기 방식 기억
* **서브폴더 패널 (F-35)** — 현재 폴더의 하위 폴더를 격자 형태로 표시, 클릭으로 빠른 이동; 상태바 ⊞ 버튼 또는 `Ctrl+Tab`으로 토글
* **숨김 파일 토글** — `Ctrl+H` 또는 "." 버튼
* **폰트 크기 조절** — `Alt+휠`로 실시간 조절 (7~24pt)
* **보기 모드 전환** — `Ctrl+휠` 또는 `Ctrl+1~5`
* **정렬 버튼** — 상태표시줄의 이름·크기·확장자·날짜 버튼으로 빠른 정렬 전환
* **폰트 비례 칼럼 너비** — 크기·날짜·확장자 칼럼이 폰트 크기(11pt 기준)에 비례해 확대/축소됨. 폰트를 키우면 칼럼도 넓어져 글씨가 잘리지 않고, 줄이면 좁아짐. 이름 칼럼은 나머지 공간을 채움
* **EXT4 파티션 브라우저 (Windows 전용)** — 리눅스 사용자가 부득이 윈도우에서 작업할 때 자신의 EXT4(리눅스) 파티션 파일을 읽을 수 있음. 관리자 모드 필요, 읽기 전용. 일반 폴더 뷰와 동일하게 동작 — 모든 보기 모드(Ctrl+1~5)가 똑같이 렌더링되고, 이름·크기·날짜 정렬(Ctrl+6~8)이 폴더 우선으로 동작

### 검색
* **항상 표시 검색창** — 툴바 오른쪽에 항상 활성화
* **하위 폴더 재귀 검색** — 현재 폴더 기준 전체 하위 탐색; 결과는 폴더별 트리 구조로 표시 (폴더 노드 클릭 시 해당 폴더로 이동); 최근·별표·휴지통 뷰는 기존 평면 목록 유지
* **300ms 디바운스** — 입력 즉시 실행하지 않고 자연스럽게 시작
* **검색 중 파일 열기** — 검색 결과에서 파일을 열어도 검색 유지
* **검색 결과 드래그** — 검색 결과 파일을 다른 앱으로 드래그 가능
* **검색 결과 칼럼 레이아웃** — 이름·크기·확장자·날짜 4컬럼; 폴더 노드는 선택 불가 그룹 헤더로 표시 (위치 컬럼은 트리 구조로 대체)
* **검색 결과 썸네일 보기** — 검색 중에도 보기 모드(Ctrl+2~5)가 그대로 적용; 아이콘/썸네일 보기에서는 폴더명이 전체 폭 한 줄 헤더로, 파일은 썸네일로 그룹 정렬되어 표시 (복사·이동·삭제·미리보기·드래그 모두 동일하게 동작)

### 파일 작업
* **복사 (F7)** — 즐겨찾기·장치 사이드바가 포함된 대상 폴더 선택 다이얼로그; 대기 중인 항목은 X 버튼으로 목록에서 제거 가능
* **이동 (F6)** — 즐겨찾기·장치 사이드바가 포함된 대상 폴더 선택 다이얼로그
* **삭제 (Del)** — 확인 대화상자 후 삭제; `Shift+Del`은 확인 없이 즉시 삭제
* **이름 변경 (F2)** — 인라인 편집, 확장자 제외 이름만 자동 선택
* **일괄 이름 변경 (F10)** — 접두사·접미사·찾기바꾸기·번호부여 다이얼로그
* **새 폴더 (Ctrl+N)** — 어디서나 (사이드바 포커스 포함)
* **패턴으로 선택 (Ctrl+Shift+S)** — 현재 폴더에서 와일드카드 패턴으로 파일 선택; `*` `?` 없으면 자동으로 `*패턴*` (contains) 처리; 우클릭 메뉴에서도 실행 가능
* **선택 항목으로 새 폴더** — 파일 2개 이상 선택 후 우클릭 → "선택 항목으로 새 폴더"; 폴더명 입력 후 선택한 파일이 새 폴더로 이동; Ctrl+Z 복구 가능
* **되돌리기 (Ctrl+Z)** — 파일 만들기·삭제·복사·이동·일괄이름변경 되돌리기 (최대 20단계)
* **템플릿으로 새 파일** (Linux) — 빈 공간 우클릭 → "New File from Template" 하위메뉴; `~/Templates/` 파일 목록으로 새 파일 생성; 동명 파일은 자동 번호 부여 `file (1).ext`; Ctrl+Z 복구 가능
* **잘라내기 (Ctrl+X)** — 파일을 반투명으로 표시; 붙여넣기 시 이동
* **복사 (Ctrl+C)** — 파일을 클립보드에 복사
* **붙여넣기 (Ctrl+V)** — 잘라내기한 파일은 이동, 복사한 파일은 복제
* **다른 앱으로 열기** — MIME 기반 앱 목록 + 추천 앱; 마지막 사용 앱 기억
* **파일 뷰어 (F3)** — 텍스트·이미지 전용 읽기 전용 내장 뷰어; 최대화 버튼 지원
* **파일 편집기 (F4)** — 내장 텍스트 편집기; 선택 없이 F4 누르면 새 파일 생성; 저장 후 즉시 목록 반영
* **파일 비교 (Ctrl+D)** — 두 번째 파일 선택 후 File A / File B / Diff 3탭 비교 뷰
* **빠른 미리보기 (F11)** — 선택 파일의 이미지·텍스트를 슬라이드인 패널로 표시
* **터미널 열기 (F9)** — 현재 폴더에서 터미널 실행; 툴바 버튼 우클릭으로 선호 터미널 앱 변경
* **폴더 동기화+백업 (F8)** — 단방향 백업(왼쪽→오른쪽) 또는 양방향 동기화(최신 파일 상호 복사·삭제); 미리보기 테이블 후 실행; 2창 모드 전용
* **드래그 앤 드롭** — 기본=이동, Alt+드래그=복사, Shift+드래그=복제(자동 이름 증가), 우클릭 드래그=컨텍스트 메뉴(이동/복사/바로가기/취소); 파일·폴더 하나만 끌면 복사/이동 다음에 "바로가기"(심볼릭 링크) 항목 표시; 드래그 중 커서 피드백
* **속성** — 파일 이름·경로·크기·수정일; 폴더는 직접 파일 수 + 전체 재귀 파일 수 표시; 권한(chmod) 변경 가능
* **압축 (Ctrl+우클릭 → 압축...)** — zip·tar.gz·tar.bz2·tar.xz 형식 지원
* **압축 해제** — 아카이브 더블클릭 시 이름 폴더 생성 후 자동 해제·이동
* **장바구니 (Alt+B)** — 여러 폴더를 오가며 파일을 담고(우클릭 → "장바구니에 담기" 또는 `Alt+B`), 상태바 🧺 버튼 팝업에서 압축·복사·이동·삭제를 한 번에 실행; 디스크에서 사라진 항목은 자동 정리
* **Scripts 하위메뉴** (Linux) — 우클릭 → "Scripts" 하위메뉴에 `~/.local/share/nautilus/scripts/` 실행 파일 목록 표시; 선택 파일 경로가 Nautilus 호환 환경변수로 전달됨

### 툴바 사용자정의 버튼
* **사용자정의 버튼 3개** — 제목 표시줄의 터미널 버튼 옆에 위치
* **우클릭으로 설정** — 각 버튼에 이름·명령어·아이콘 지정; 아이콘은 `.desktop` 파일에서 자동 로드 및 저장
* **앱 선택 다이얼로그** — 설치된 앱 전체를 검색하여 선택 (표준 경로 + Snap + Flatpak); 명령어에 `%f`를 넣으면 실행 시 선택된 파일 경로로 치환

### 시스템
* **창 크기·위치 저장** — 닫으면 저장, 다음 실행 시 복원
* **즐겨찾기 저장** — OS 표준 설정 경로에 저장 (Windows: %APPDATA%, Linux: ~/.config)
* **실시간 파일 감지** (Linux) — inotify 기반, 새 파일 즉시 목록 반영
* **44개 언어 UI** — 시스템 로케일에 맞는 자동 번역 인터페이스
* **기본 파일 관리자 설정** — 타이틀바 앱 아이콘 클릭 → 팝업의 "기본앱으로" 버튼 한 번으로 `xdg-mime` + `gsettings` 자동 등록; 이미 설정된 경우 초록색 "✓ 기본앱 설정됨"으로 표시
* **시스템 전체 파일 열기/저장 대화상자 (Windows)** — 파일 대화상자 선택기를 등록하면 **모든** 앱의 파일 열기 **및 저장** 대화상자가 Windows 기본 창 대신 파일이야기로 열립니다 — Chrome, Edge, PDF이야기, 대부분의 Win32/Electron 앱 포함. 사용자 권한(HKCU)으로 등록되어 관리자 권한 불필요. 탐색기 우클릭 통합과는 별도 토글이라 이것만 따로 끌 수 있음

## 🚀 설치

### Linux — zip (모든 배포판)

1. `FileIyagi-vX.X.X-linux-x64.zip` 다운로드 후 압축 해제
2. 실행:
   ```bash
   bash install.sh
   ```
   바이너리를 `~/.local/bin/`에 설치하고, 아이콘과 `.desktop` 파일을 등록하며, 기본 파일 관리자로 설정하고, 브라우저 저장 대화상자를 위한 XDG Portal FileChooser 백엔드를 구성합니다.

**Qt6 필요** — 없을 경우 설치:

| 배포판 | 명령어 |
|--------|--------|
| Ubuntu / Debian | `sudo apt install libqt6core6 libqt6widgets6 libqt6sql6 libqt6dbus6` |
| Fedora | `sudo dnf install qt6-qtbase qt6-qtsql` |
| Arch Linux | `sudo pacman -S qt6-base` |
| openSUSE | `sudo zypper install libqt6-qtbase` |

### Linux — deb (Ubuntu / Debian)

```bash
sudo dpkg -i fileiyagi_X.X.X_amd64.deb
```

### Windows

[Microsoft Store](https://apps.microsoft.com/search?query=FileIyagi)에서 이용 가능합니다.

## 🖧 썸네일 설정

동영상 썸네일을 사용하려면 `ffmpegthumbnailer`를 설치하세요.

```bash
# Linux
sudo apt install ffmpegthumbnailer
```

Windows에서 동영상 썸네일을 사용하려면 `ffmpeg`를 설치하세요 (`C:\ffmpeg\bin\` 또는 PATH).

PDF 썸네일은 Linux에서 `poppler-qt6`, Windows에서 Qt6::Pdf를 사용하여 자동 지원됩니다.

- **저장 대화 상자 자동 이름 증가** — 저장 대화 상자에서 파일을 클릭하면 다음 파일 이름이 자동으로 제안됩니다(`abc.pdf` → `abc2.pdf`, `abc9.pdf` → `abc10.pdf`). 확장자는 자동으로 유지됩니다.
